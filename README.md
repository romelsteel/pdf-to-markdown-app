# PDF → Markdown

Convert PDFs (and Word, PowerPoint, Excel, images, HTML) into clean **Markdown**
you can paste straight into an AI. A wolf-themed 🐺 desktop app for Windows,
powered by [Microsoft MarkItDown](https://github.com/microsoft/markitdown).

➡️ **[Open the app &amp; full documentation](pdf-to-markdown/README.md)** · [🇨🇿 Česky](pdf-to-markdown/README.cs.md)

**Highlights:** drag-and-drop, batch convert, automatic clean-up of typical PDF
artifacts, dark mode, Czech/English interface, copy / save / zip, optional
"split for AI" chunking, and per-file character/token counts. Runs entirely
locally — your files never leave your machine.

```bash
cd pdf-to-markdown
py -m pip install -r requirements.txt
py app.py
```

…or build a single double-click `.exe` with `build_exe.bat`. See the
[full README](pdf-to-markdown/README.md) for details.
